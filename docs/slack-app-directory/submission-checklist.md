## Short description (<150 chars)
```
Morgenruf is a free, open-source standup bot. Daily check-ins, analytics, AI summaries — self-hosted or cloud.
```

## Long description
```
Morgenruf is a free, open-source Slack standup bot that helps remote and hybrid teams stay aligned with daily async standups.

**Key Features:**
• 🌅 Automated daily standups — bot DMs each team member at your configured time
• 📊 Analytics dashboard — participation trends, response history, CSV export
• 🤖 AI summaries — optional GPT/Claude-powered daily summary posted to your channel
• 📅 Multiple schedules — different standup configs per team or channel
• 🏆 Kudos system — peer recognition with leaderboard
• ⚡ Automation rules — notify manager on blockers, reminder on low participation
• 🔗 Jira/GitHub/Linear autolink — ticket numbers become clickable links
• 🔧 MCP server — expose standup data to AI assistants like Claude and Cursor
• 25+ question templates — engineering, sales, retro, OKR, and more

**Why Morgenruf?**
Unlike Geekbot ($8/user/mo) or DailyBot ($3.60/user/mo), Morgenruf is completely free and open-source. Self-host on your own infrastructure for full data ownership, or use our hosted version.

**Get started in under 2 minutes:**
1. Click "Add to Slack"
2. Choose your standup channel and time
3. Team members get DMs automatically at the scheduled time

No credit card. No usage limits. No data lock-in.
```

## Submission steps
1. Go to api.slack.com/apps → select Morgenruf app
2. Click "Distribute App" in left sidebar
3. Complete "App Distribution Checklist"
4. Click "Submit for Review"
5. Review takes 1–2 weeks

## Privacy Policy (needs to be created at morgenruf.dev/privacy)
Key points to cover:
- What data is stored (standup responses, Slack user IDs, workspace metadata)
- Where data is stored (self-hosted: your servers; cloud: morgenruf.dev servers)
- Data retention policy
- How to delete data (contact hello@morgenruf.dev or self-hosted: drop DB)
- GDPR compliance statement
