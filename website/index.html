How it works

Up and running in minutes

Click “Add to Slack” and authorize the bot in your workspace. Takes less than 30 seconds.

⚙️

Self-host

Your infra. Your data.

Don’t want to trust the hosted version? Self-host Morgenruf in under 5 minutes with Docker, or deploy to Kubernetes with our Helm chart.

  • ✓ Full control over your standup data
  • ✓ Deploy to any cloud or on-premises
  • ✓ Kubernetes-ready with Helm chart
  • ✓ Detailed setup docs on GitHub
$ docker run -d \
--name morgenruf \
-e SLACK_BOT_TOKEN=xoxb-... \
-e SLACK_SIGNING_SECRET=... \
-e DATABASE_URL=postgres://... \
-p 3000:3000 \
ghcr.io/morgenruf/morgenruf:latest
# That's it. Your bot is live. 🌅
$ helm repo add morgenruf \
https://morgenruf.github.io/charts

$ helm install morgenruf \
morgenruf/morgenruf \
--set slack.botToken=xoxb-... \
--set slack.signingSecret=...
# Kubernetes-native. Scales to zero. 🚀
