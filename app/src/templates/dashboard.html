🔌 MCP — Connect AI Assistants
Let Claude, Cursor, and GitHub Copilot query your standup data via https://api.morgenruf.dev/mcp
API Keys
Client Configuration

Add to ~/Library/Application Support/Claude/claude_desktop_config.json

{
  "mcpServers": {
    "morgenruf": {
      "url": "https://api.morgenruf.dev/mcp",
      "headers": {
        "Authorization": "Bearer mrn_YOUR_KEY_HERE"
      }
    }
  }
}

Add to ~/.cursor/mcp.json

{
  "mcpServers": {
    "morgenruf": {
      "url": "https://api.morgenruf.dev/mcp",
      "headers": {
        "Authorization": "Bearer mrn_YOUR_KEY_HERE"
      }
    }
  }
}

Add to .vscode/mcp.json (workspace) or user settings

{
  "servers": {
    "morgenruf": {
      "type": "http",
      "url": "https://api.morgenruf.dev/mcp",
      "headers": {
        "Authorization": "Bearer mrn_YOUR_KEY_HERE"
      }
    }
  }
}

Raw HTTP — useful for testing or custom integrations

curl -X POST https://api.morgenruf.dev/mcp \
  -H "Authorization: Bearer mrn_YOUR_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/list","id":1}'
